Fine-Tuning Neural Network for Sketch-Aware Object Segmentation

Fine-tuned Meta’s Segment Anything Model (SAM) on the OpenSketch dataset to improve flood-fill accuracy in raster sketches using bounding-box prompts, decoder-only training, and a custom BCE + Dice loss. Improved metrics from baseline (Pixel Accuracy: 8.85%, mIoU: 1.74%, F1: 3.22%) to 95.46%, 78.60%, and 87.65%.
